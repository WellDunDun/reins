## [0.1.0] - 2026-02-22

### Added
- `reins init` — scaffold full harness engineering structure
- `reins audit` — score repos against six dimensions (0-18)
- `reins evolve` — roadmap to next maturity level
- `reins doctor` — health check with prescriptive fixes
- Five-level maturity model (L0 Manual through L4 Self-Correcting)
- Claude Code skill for AI-native integration
- CI pipeline with lint, test, typecheck, and self-audit
- Zero external runtime dependencies
